Available CRUD API Methods:

[Method] - [path to API call] → [Info]

GET - /   [this page]

GET - /app   [Phone Book App] → There are more methods associated with this app (i.e. JS, CSS, and manifest) but they are unnecessary to mention.

GET - /api/allContacts → Retrieves all contacts from Database

POST - /api/addContact → Allows you to add a contact to Database and returns the added contact with ID

PUT - /api/updateContact → Allows you to update a contact from Database given an ID of the contact

DELETE - /api/delete/contact → Allows you to delete a contact from Database given an ID of the contact
